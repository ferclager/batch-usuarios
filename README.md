# chore/02-tasklet - Tu primer Job de Spring Batch: Tasklet y metadata

## Descripción
Esta branch corresponde al tercer video de la playlist **Spring Batch desde 0: Procesamiento por Lotes con Spring Boot 3.5**.

En este capítulo se explica:
- Cómo crear tu primer Job de Spring Batch
- Uso de Tasklets para procesamiento simple
- Metadata de Spring Batch y su importancia
- Configuración básica de Jobs y Steps
- Ejecución y monitoreo de Jobs
