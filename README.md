# chore/01-initialzr - Crea tu proyecto Spring Batch desde cero con Spring Initializr

## Descripción
Esta branch corresponde al segundo video de la playlist **Spring Batch desde 0: Procesamiento por Lotes con Spring Boot 3.5**.

En este capítulo se explica:
- Cómo crear un proyecto Spring Batch usando Spring Initializr
- Configuración inicial del proyecto
- Dependencias necesarias para Spring Batch
- Estructura básica del proyecto
