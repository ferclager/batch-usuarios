# batch-usuarios

Este es el enlace de la playlist [Spring Batch desde 0: Procesamiento por Lotes con Spring Boot 3.5](https://www.youtube.com/playlist?list=PLETWWcGiZcBs) de Youtube.
## Descripción
Aprende Spring Batch 5 desde cero construyendo un proyecto real que importa un CSV de usuarios a una base de datos. 
Dominarás los conceptos clave del
procesamiento por lotes: Jobs, Steps, Chunks, ItemReaders, ItemProcessors, ItemWriters, tolerancia a fallos, reanudación, escalado, programación y
testing. 
Stack: Spring Boot 3.5, Spring Batch 5, Java 21, Gradle y H2.
